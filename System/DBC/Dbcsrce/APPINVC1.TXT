.

                   *P25:20,"__",*HOFF,*P25:20,INDEX

. BRANCH TO THE ROUTINE INDICATED BY THE INDEX

         TRAP      BADCHAIN IF CFAIL
         CLOCK     TIME TO TIME
         BRANCH    INDEX OF NCLINVCS:  ENTER A NEW ORDER
                            LIBLIST:  LIST  LIBERTY OPEN INVOICE
                            DARLIST: PRINT DAILY INVC TRANS LIST
                            ALFARLST: PRINT DAILY INVC TRANS BY ACCT
                            SOENTRY:  ENTER PACKING LIST DATA
                            SOPRINT:  PRINT PACKING LIST
                            SOINVPRT: PRINT INVOICE MAILERS
                            SOUPDATE: MOD/DELETE PKG LIST PRIOR TO INVOICING
                            LIBMENU1   SELECT NEW PROCESSING SYST
         GOTO      GETINDEX
